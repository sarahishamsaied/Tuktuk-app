
# TukTuk App

An upcoming simple NextJs app to allow users to book any TukTuk around the world!

## How to start the project

First, run the development server:

```bash
npm run dev
```
or

```
yarn dev
```


```
Default PORT : 3000
```

## Tools & Dependencies

1 - NextJs

2 - Firebase

3 - TailwindCss
